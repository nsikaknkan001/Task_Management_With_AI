## 🔑 Key Features

### 🌟 Core Features
- **User Authentication:** Securely register and confirm your account via email.
- **Task Management:** Create, read, update, and delete tasks with ease.
- **Task Prioritization:** Assign priority levels to tasks: High, Medium, Low.
- **AI Roadmap Generation:** Use Google Gemini AI to automatically generate a roadmap for your tasks.
- **Real-Time Notifications:** Get reminders about your tasks when they’re due.
- **Responsive Design:** Enjoy a modern, mobile-friendly interface built with Angular and Material Design.

### 🤖 AI Integration
- Experience intelligent roadmap generation through Google Gemini AI.
- The application will handle any service interruptions to ensure your tasks stay managed.
- Get task recommendations based on the details you provide.

### 🔐 Security Features
- Your data is secure with JWT-based authentication.
- The application is configured to allow safe cross-origin requests.
- An email verification process adds an extra layer of protection.
- User passwords are securely hashed with BCrypt for enhanced security.

## 💻 Technology Stack

### Frontend
- **Angular 20.0.0:** A robust framework for building user interfaces.
- **Angular Material:** A UI component library that follows Material Design guidelines.
- **TypeScript:** A superset of JavaScript that adds strong typing.
- **RxJS:** A library for reactive programming using Observables.

### Backend
- **Spring Boot 3.2.0:** A Java-based framework that simplifies backend development.
- **RESTful API:** Efficient communication between the frontend and backend.

## 📊 System Requirements

To run Task Management With AI smoothly, your system should meet the following requirements:

- **Operating System:** Windows 10 or later, macOS (version 10.13 or later), or a modern Linux distribution.
- **RAM:** At least 4 GB.
- **Storage:** Minimum of 1 GB of available space.
- **Internet Connection:** Required for account registration and AI functions.
